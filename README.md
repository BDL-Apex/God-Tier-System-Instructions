​🚀 Overview (ภาพรวม)
​Repository นี้รวบรวม System Instructions (คำสั่งระบบ) ระดับสูงที่ถูกออกแบบมาเพื่อรีดประสิทธิภาพสูงสุดจาก Large Language Models (LLMs) โดยเฉพาะ Gemini สถาปัตยกรรมนี้เน้นการตัดส่วนเกิน (Zero Fluff) และความแม่นยำทางตรรกะ
​💎 Core Features (คุณสมบัติหลัก)
​Zero-Latency Output: ตัดคำเกริ่นนำและบทสนทนาที่ไม่จำเป็น มุ่งเน้นผลลัพธ์ทันที
​Precision Reasoning: ใช้ Chain of Thought (CoT) ขั้นสูงสำหรับงานตรรกะและ Coding
​Hallucination Control: มีโปรโตคอลตรวจสอบข้อเท็จจริงและบังคับใช้กฎ "Insufficient Data" อย่างเคร่งครัด
​Adaptive Personas: ระบบเปลี่ยนโหมดอัตโนมัติ (Coding, Strategy, Creative) ตามบริบท
​🛠 Usage (วิธีการใช้งาน)
​ไปที่ไฟล์ในโฟลเดอร์ /prompts (หรือเลือกไฟล์ System Prompt หลัก)
​คัดลอกเนื้อหาทั้งหมด
​นำไปวางในส่วน System Instructions ของ AI Studio หรือผ่าน API
​เริ่มต้นใช้งานได้ทันที
​🧠 Philosophy: The CO-STAR Framework
​Prompt ทั้งหมดในโปรเจกต์นี้ถูกสร้างขึ้นตามมาตรฐาน CO-STAR:
​C (Context): บริบทชัดเจนระดับสูง
​O (Objective): เป้าหมายที่วัดผลได้
​S (Style): สไตล์การเขียนที่แม่นยำ
​T (Tone): น้ำเสียงที่เหมาะสมกับงาน (เช่น Hyper-intelligent)
​A (Audience): กลุ่มเป้าหมายเฉพาะเจาะจง
​R (Response): รูปแบบผลลัพธ์ที่กำหนดโครงสร้างไว้ชัดเจน (Markdown, JSON, Code)
